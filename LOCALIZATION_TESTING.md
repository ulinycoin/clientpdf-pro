# 🧪 Локальное тестирование русской локализации

## 🚀 Быстрый старт

### 1. Переключение на ветку локализации
```bash
git checkout feature/russian-localization
```

### 2. Установка зависимостей
```bash
npm install --legacy-peer-deps
```

### 3. Запуск development сервера
```bash
npm run dev
```

## 🔧 Способы тестирования

### Метод 1: Интеграция в существующие компоненты
Добавьте в любую существующую страницу:

```typescript
// Добавьте в начало файла
import useLocalization from '@/hooks/useLocalization';
import LanguageSwitcher from '@/components/atoms/LanguageSwitcher';

// В компоненте
const { t } = useLocalization();

// В JSX
<div>
  <LanguageSwitcher />
  <h1>{t('header.title')}</h1>
  <p>{t('header.subtitle')}</p>
</div>
```

### Метод 2: Тестирование демо-страницы
Создайте временный роут или импортируйте компонент:

```typescript
import LocalizationDemo from '@/components/pages/LocalizationDemo';

// Используйте в любом месте приложения
<LocalizationDemo />
```

### Метод 3: Тестирование отдельных компонентов
```typescript
// Тестирование переключателя языков
import LanguageSwitcher from '@/components/atoms/LanguageSwitcher';

// Разные варианты
<LanguageSwitcher variant="dropdown" showText={true} />
<LanguageSwitcher variant="toggle" showText={false} />
```

## 🐛 Общие проблемы и решения

### Проблема: "Cannot resolve '@/hooks/useLocalization'"
**Решение**: Убедитесь, что path aliases настроены в tsconfig.json и vite.config.ts

### Проблема: Переводы не отображаются
**Решение**: 
1. Проверьте, что JSON файлы корректны (нет синтаксических ошибок)
2. Убедитесь, что ключи перевода существуют в обоих языках
3. Проверьте console на наличие ошибок

### Проблема: Язык не сохраняется
**Решение**: Проверьте, что localStorage доступен (не в приватном режиме браузера)

### Проблема: Некорректное форматирование чисел
**Решение**: Убедитесь, что браузер поддерживает Intl.NumberFormat для нужной локали

## 🎯 Примеры использования

### Простой перевод
```typescript
const { t } = useLocalization();
return <h1>{t('tools.merge.title')}</h1>; // "Merge PDF" или "Объединить PDF"
```

### Перевод с параметрами (будущее расширение)
```typescript
const { t } = useLocalization();
return <p>{t('common.filesSelected', { count: 3 })}</p>; // "3 files selected" или "3 файла выбрано"
```

### Форматирование утилит
```typescript
const { formatFileSize, formatNumber } = useLocalization();
return (
  <div>
    <p>Размер: {formatFileSize(1048576)}</p> {/* "1.0 MB" или "1,0 МБ" */}
    <p>Страниц: {formatNumber(1234)}</p> {/* "1,234" или "1 234" */}
  </div>
);
```

### Переключатель в Header
```typescript
// Замените существующий Header на HeaderLocalized
import HeaderLocalized from '@/components/organisms/HeaderLocalized';

<HeaderLocalized />
```
